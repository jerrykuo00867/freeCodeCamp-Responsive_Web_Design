Pengenalan HTML

HTML atau HyperText Markup Language adalah bahasa markup standar yang digunakan untuk membuat dan menyusun struktur halaman web. HTML bukan bahasa pemrograman, melainkan bahasa yang digunakan untuk memberi tahu browser bagaimana sebuah konten harus ditampilkan melalui sistem tag.

Struktur Dasar HTML

Setiap dokumen HTML wajib memiliki struktur dasar yang terdiri dari deklarasi tipe dokumen, tag pembuka, bagian kepala untuk metadata, dan bagian tubuh untuk konten.

Elemen Teks

HTML menyediakan berbagai tag untuk menstrukturkan teks agar memiliki hierarki yang jelas.

  • Heading (Judul)

    • Heading digunakan untuk membuat judul dan subjudul, dimulai dari H1 (terbesar/terpenting) hingga H6 (terkecil).

  • Paragraf

    • Paragraf digunakan untuk menulis teks blok biasa. Browser secara otomatis akan memberikan jarak baris baru sebelum dan sesudah tag ini.

Referensi Tag Penting

  • Tag HTML: Pembungkus utama seluruh dokumen web.
  • Tag Head: Wadah untuk metadata yang tidak ditampilkan langsung di halaman web, seperti judul tab dan tautan file eksternal.
  • Tag Body: Wadah untuk semua konten visual yang akan dilihat oleh pengguna, seperti teks, gambar, dan video.
