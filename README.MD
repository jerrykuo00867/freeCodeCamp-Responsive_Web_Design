# freeCodeCamp-Responsive Web Design Certification

Repositori ini berisi kumpulan projek yang saya selesaikan semasa mengikuti kurikulum freeCodeCamp. Projek ini berfungsi sebagai bukti pembelajaran, perkembangan kemahiran coding, dan jurnal perjalanan saya dalam pembangunan web.

## Projek

Berikut adalah projek yang terdapat dalam repositori ini:

* **Audio Video Player**
* **Cat Blog Page**
* **Cat Photo App**
* **Coding Journey Blog Page**
* **Final Exam Table**
* **Fried Rice Recipe**
* **Heart Icon**
* **Introduction**
* **List Major Website**
* **Music Player**
* **Quincy's Job Tips Page**
* **Travel Agency Page**
* **Video Player**
* **Video Compilation Page**
* **XYZ Bookstore**
* **iframe Video Display**

## Cara Menjalankan Projek

1. Klon repositori ini ke komputer anda.
2. Buka mana-mana folder projek di atas.
3. Klik dua kali pada file `index.html` untuk melihat hasilnya di pelayar web (browser).
